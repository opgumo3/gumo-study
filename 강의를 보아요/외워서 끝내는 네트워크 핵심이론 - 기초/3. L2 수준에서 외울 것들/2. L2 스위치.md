# L2 Access switch
- l2 스위치
- 엔드 포인트와 직접 연결되는 스위치
- mac주소로 스위치

- 스위치는 여러 기기와 연결할 수 있는데, 그걸 Port/Interface 라고 표현.
- 그리고 여러개 모아져 있으니 hub 라는 말도 붙인다.
- 24 port 스위치 허브 ..

- Link up : L2 Access 까지 연결 잘 됐다.  <-> Link down (물리적으로 케이블 제거)
- L2 랑 Router(L3) 랑 이어지는 건 Uplink
