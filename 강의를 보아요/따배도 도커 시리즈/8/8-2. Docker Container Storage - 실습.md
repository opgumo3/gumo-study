# -v 실습

### 1. 익명 볼륨 생성
```sh
docker run -d -v /var/lib --name nginx nginx

# 생성된 Volume 확인
docker volume ls
docker volume inspect [volume_name] # Mountpoint : /var/lib/docker/volumes/[uuid]/_data
docker container inspect [container_name] # Mounts 

docker rm -f nginx
docker volume prune # 사용하지 않는 volume 삭제
```

### 2. host 경로에 마운트
```sh
docker run -d --name mysql -e MYSQL_ROOT_PASSWORD=1234 -v /Users/me/dbdata:/var/lib/mysql mysql:8.0

docker rm -f mysql # 컨테이너가 삭제되어도 데이터는 남아있음.
```
