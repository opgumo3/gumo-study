# 6.1 표현 영역과 응용 영역
`표현 영역`과 `응용 영역`은 도메인 영역과 사용자를 연결해주는 매개체 역할을 한다. </br>
각 영역이 어떤 역할을 수행하는지 다시 알아보자.

### 표현 영역
- 사용자의 요청을 해석하고, 사용자가 원하는 기능을 제공하는 응용 서비스를 실행한다.
- 사용자로부터 전달받은 데이터와 응용 서비스가 요구하는 파라미터는 서로 다르기 때문에, 사용자의 요청을 변환한다.
- 응용 서비스의 실행 결과를 사용자에게 알맞은 형식으로 응답한다

### 응용 영역
- 실제 사용자가 원하는 기능을 제공한다.
- 표현 영역에 의존하지 않고 기능 실행에 집중한다.
