# 4.4 애그리거트 로딩 전략
리포지터리에서 애그리거트를 조회한 시점에 애그리거트는 완전한 상태여야 한다.
이를 위해 조회 방식을 `즉시 로딩`으로 설정할 수 있다.
@Entity 나 컬렉션의 fetchType 을 FetchType.EAGER 로 설정하면 된다.

```java
pubic class Product {
    ...

    @OneToMany(fetch = FetchType.EAGER)
    @JoinColumn(name = "product_id")
    @OrderColumn(name = "list_idx")
    private List<Image> images = new ArrayList<>();

    ...
}
```
